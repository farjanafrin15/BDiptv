# কন্ট্রিবিউটিং গাইডলাইন

আপনি যদি নতুন চ্যানেল যুক্ত করতে চান বা বিদ্যমান লিংক আপডেট করতে চান, নীচের ধাপগুলো অনুসরণ করুন:

1. রিপো Fork করুন অথবা সরাসরি একটি নতুন ব্রাঞ্চ তৈরি করুন: `git checkout -b add-channel-yourname`
2. `iptv.m3u8`-এ নতুন এন্ট্রি যোগ করুন (নিম্নলিখিত ফরম্যাট অনুসরণ করুন) অথবা আলাদা ফাইল যুক্ত করুন।
   উদাহরণ এন্ট্রি:
   ```
   #EXTINF:-1 tvg-id="NTV" tvg-name="NTV" group-title="Bangla",NTV
   http://example.com/ntv-stream.m3u8
   ```
3. পরিবর্তন পুশ করুন এবং একটি Pull Request খুলুন। PR-এ উল্লেখ করুন লিংক সোর্স ও টেস্ট স্টেটাস।
